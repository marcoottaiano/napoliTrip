# Napoli Trip Showcase

Setup iniziale di un sito vetrina in **Next.js** per raccontare un viaggio di coppia a Napoli.

## Stack scelto

- Next.js (App Router + TypeScript)
- Tailwind CSS
- Framer Motion
- GSAP + ScrollTrigger (previsto per le prossime iterazioni)
- Lenis (smooth scrolling, prossima integrazione)

## Avvio

```bash
npm install
npm run dev
```

## Roadmap breve

1. Definizione struttura storytelling (hero, itinerario, moodboard, timeline giornaliera).
2. Integrazione scroll effects (GSAP ScrollTrigger + sezioni pin/reveal).
3. Parallasse multi-layer su hero e su sezioni fotografiche.
4. Ottimizzazione responsive e rifiniture cinematiche (blur, gradients, light leaks).
